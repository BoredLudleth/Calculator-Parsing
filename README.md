## Часть I. LR(0) грамматика нашего языка ##
```
G   ::= E<EOF>
E   ::= E{[+-]T}*
T   ::= T{[*/]P}*
P   ::= (E)|ID
ID  ::= VAR|NUM
VAR ::= [xyz]
NUM ::= [0-9]*
```
